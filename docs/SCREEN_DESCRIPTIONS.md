# Screen Interface Descriptions

## Splash Screen Interface
This interface displays the app logo and loading animation when users first open the application. The screen shows the Asir Honey Marketplace branding with a smooth transition to the onboarding screens.

## Onboarding Interface
This interface introduces new users to the app features through three informative slides. Users can swipe through the slides to learn about the marketplace, then tap the "Get Started" button to proceed to account type selection.

## Account Type Selection Interface
This interface allows users to choose their account type before registration. Users can select either "Consumer" to buy honey products or "Beekeeper" to sell their products, then tap "Continue" to proceed to registration.

## Create Account Interface
This interface is designed to create a user account on the app. Users can enter all required information, including their name, email, password, phone, and account type, then click the sign up button to create their account.

## Login Interface
This interface enables existing users to access their accounts. Users enter their registered email and password, then tap the "Login" button to authenticate and access the main application features.

## Consumer Home Interface
This interface displays the main marketplace for consumers. Users can browse featured products, view categories, search for honey types, and access their cart and profile through the navigation menu.

## Product Details Interface
This interface shows comprehensive information about a selected honey product. Users can view product images, read descriptions, check ratings and reviews, select quantity, and add items to their cart.

## Shopping Cart Interface
This interface displays all items added to the user's cart. Users can review selected products, modify quantities, view total prices, and proceed to checkout by tapping the "Checkout" button.

## Checkout Interface
This interface handles the order completion process. Users enter their delivery address, select payment method, review order summary, and confirm their purchase by tapping the "Place Order" button.

## Order History Interface
This interface shows all previous orders made by the consumer. Users can view order status, track deliveries, and access order details by tapping on individual order items.

## Beekeeper Dashboard Interface
This interface provides beekeepers with business overview and statistics. Users can view total products, orders, revenue, and ratings, then navigate to product management or order processing sections.

## Manage Products Interface
This interface allows beekeepers to view and manage their honey products. Users can see their product list, edit existing items, delete products, and add new products using the floating action button.

## Add Product Interface
This interface enables beekeepers to create new product listings. Users can upload product images, enter product details including name, description, price, and stock, then save the product to their catalog.

## Beekeeper Orders Interface
This interface displays all orders containing the beekeeper's products. Users can view order details, update order status, and manage customer orders through different status tabs.

## Categories Interface
This interface displays different honey categories for easy browsing. Users can view honey types including Sidr, Acacia, Wildflower, and Mixed varieties, then tap on any category to view related products.

## Product List Interface
This interface shows all products within a selected category or search results. Users can browse through product cards, view ratings and prices, and tap on any product to access detailed information.

## Search Interface
This interface enables users to search for specific honey products. Users can enter search terms, filter by category or price range, and view matching results in a organized list format.

## Favorites Interface
This interface displays all products marked as favorites by the user. Users can view their saved products, remove items from favorites, and quickly access product details for future purchases.

## Notifications Interface
This interface shows all app notifications including order updates and promotional messages. Users can view notification details, mark items as read, and manage their notification preferences.

## Edit Product Interface
This interface allows beekeepers to modify existing product information. Users can update product images, change descriptions, adjust prices and stock levels, then save changes to their product catalog.

## Order Details Interface
This interface provides comprehensive information about a specific order. Users can view order items, delivery status, payment details, and tracking information for their purchases.

## Profile Interface
This interface allows users to manage their account information. Users can update personal details, change passwords, switch app language and theme, and access app settings and support options.
